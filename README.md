## Aplicație Nativă Android - Versiunea 1.1

Jellyseerr este acum disponibil ca aplicație nativă pentru dispozitivele Android. Aceasta este o aplicație dedicată, nu un simplu wrapper web, oferind o experiență optimizată pentru mobil.

### Caracteristici Versiunea 1.1:

- **Sistem complet de update automat** - Aplicația verifică automat și se actualizează instant în interiorul aplicației fără reinstalare manuală
- **Splash Screen** - La pornire, aplicația afișează un Splash Screen cu logo animat cu efect de Zoom și background gradient (#111827 → #1E1B4B)
- **Animații** - Logo entrance animation cu pulsare smooth, crossfade animations între pagini, și Zoom effects pentru logo la ieșire
- **Design gradient modern** - Interfață cu gradient system (#111827 → #1E1B4B, #6D28D9 → #8B5CF6) și theme mov/purple consistent
- **Optimizare Aplicatie Android** - Hardware acceleration forțat, high refresh rate (120Hz dacă suportat), cache optimizat pentru încărcare rapidă
- **Fix bug navigare Samsung** - Rezolvată problema unde butonul back scotea utilizatorul complet din aplicație; acum navighează corect o pagină înapoi
- **Sistem notificări** - La prima instalare, utilizatorul primește mesaj pentru activarea notificărilor
- **Fonturi Inter integrate** - Fonturi premium Inter implementate corect cu lowercase și underscore
- **Gradient Background Splash** - Splash screen cu gradient diagonal premium în loc de fundal simplu negru
- **Gestiune permisiuni Android 13+** - Sistem complet cu ActivityResultLauncher pentru permisiuni notificări
- **SSL handling securizat** - Custom SSL error handling pentru domeniile adelin.org și adelinx.go.ro
- **Performance optimizări** - Eliminarea delay-urilor redundante, loading parallel, și memory management îmbunătățit
- **Compatibilitate extinsă** - Suport de la Android 5.0 (API 21) până la Android 16
- **UI/UX improvements** - Interfață modernizată cu efect de liquid glass blur, background transparent, margini rotunjite, butoane redesign cu fade la imagini și animații accelerate pentru o experiență mai rapidă
- **Bug fixes complete** - Rezolvate probleme de duplicate onPageFinished, missing imports, și memory leaks
- **Session management** - DOM storage pentru login persistence și cookie acceptance
- **High refresh rate support** - Optimizări pentru dispozitive cu refresh rate mai mare cum ar fi 120hz 144hz etc

2. **Permite instalarea din surse necunoscute** în setările telefonului
3. **Deschide fișierul .apk** descărcat
4. **Urmați pașii de instalare**
5. **Deschide aplicația** și configurează conexiunea la serverul tău Jellyseerr
